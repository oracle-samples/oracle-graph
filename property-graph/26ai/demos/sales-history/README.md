# Sales History Graph Studio Demo

This demo contains a Graph Studio notebook for exploring PGQL constructs with the Sales History property graph in Oracle AI Database 26ai.

Use this folder when you want to run the Sales History graph walkthrough in Graph Studio. Use [`../bank_graph/`](../bank_graph/) for the Bank Graph SQL Developer and Jupyter examples.

## What's Included

| Path | Purpose |
| --- | --- |
| [`graph-studio-notebooks/26ai-SH-property-graphs.dsnb`](graph-studio-notebooks/26ai-SH-property-graphs.dsnb) | Graph Studio notebook for the Sales History property graph walkthrough. |

## Prerequisites

- Oracle AI Database 26ai or Autonomous AI Database 26ai.
- Graph Studio access for your database environment.
- Sales History sample data available in the database.
- The Sales History property graph created and available to the notebook.

## Run The Demo

1. Open Graph Studio.
2. Import [`graph-studio-notebooks/26ai-SH-property-graphs.dsnb`](graph-studio-notebooks/26ai-SH-property-graphs.dsnb).
3. Confirm the Sales History property graph is available in your workspace.
4. Run the notebook paragraphs in order.
