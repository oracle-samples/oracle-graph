# Oracle Graph for Oracle AI Database 26ai

This folder contains property graph examples for Oracle AI Database 26ai and Autonomous AI Database 26ai. Use this path for the current Oracle Graph experience, including SQL property graph queries, Graph Studio workflows, and notebook-based demos.

If you are new to Oracle Graph, start here: [`get-started/`](get-started/).

## When To Use This Folder

Use this folder if you are working with:

- Oracle AI Database 26ai.
- Autonomous AI Database 26ai.
- SQL property graph examples using the current 26ai experience.
- 26ai SQL property graph demos and Graph Studio workflows.

If you are using Oracle Database 19c, use [`../19c/`](../19c/) instead.

## Recommended Order

1. Start with [`get-started/`](get-started/) to create your first graph and run basic queries.
2. Move to [`demos/`](demos/) for complete examples and applied scenarios.
3. Browse [`../algorithms/`](../algorithms/) when you want algorithm-specific examples.
