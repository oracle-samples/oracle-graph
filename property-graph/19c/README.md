## 19c Notes

In Oracle Database 19c, property graph workflows commonly use Oracle Graph Server and Client with PGX and PGQL. This differs from Oracle AI Database 26ai, where SQL property graph support is native in the database and Graph Studio is the primary browser-based workspace.
