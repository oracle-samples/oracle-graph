# Bank Graph Dataset

This dataset contains sample bank accounts and transfers used by the Bank Graph examples.

## Files

| File | Description |
| --- | --- |
| [`data/BANK_ACCOUNTS.csv`](data/BANK_ACCOUNTS.csv) | Sample bank account records. |
| [`data/BANK_TRANSFERS.csv`](data/BANK_TRANSFERS.csv) | Sample transfers between bank accounts. |
| [`data/LICENSE.txt`](data/LICENSE.txt) | Dataset license information. |
